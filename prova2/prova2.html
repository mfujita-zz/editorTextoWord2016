Instruções para formatação do texto da Prova 2

Realize o exercício cumprindo com as seguintes etapas:

  • Crie um novo documento no editor Word.
  • Configure o tamanho da folha para A4.
  • Configure as margens de acordo com o padrão ABNT (clique aqui para abrir em outra aba).
  • Configure a fonte através da galeria de estilos:
    Arial ou Times New Roman
    Tamanho 12
    Justificado
    Recuo esquerda: 0 cm
    Recuo direita: 0 cm
    Recuo Especial: (nenhum)
    Espaçamento antes: 0 cm
    Espaçamento depois: 0 cm
    Espaçamento entre linhas: 1,5 linhas (SIC)
  • Configure a legenda através da galeria de estilos:
    Arial ou Times New Roman
    Tamanho 10
    Centralizado
    Recuo esquerda: 0 cm
    Recuo direita: 0 cm
    Recuo Especial: (nenhum)
    Espaçamento antes: 0 cm
    Espaçamento depois: 0 cm
    Espaçamento entre linhas: simples

A ilustração a seguir mostra como ficam as configurações do estilo legenda.
