Instruções para o simulado para formatar texto

Baixe o documento que está no endereço indicado no final desta página.

Certifique-se que o tamanho da folha é A4.

Configure as margens para 3 cm em todas as bordas das folhas.

Digite os itens abaixo e separe-os em cada página (usar o recurso Quebra de página).

  • Capa
  • Folha de rosto
  • Dedicatória
  • Agradecimentos
  • Resumo
  • Lista de ilustrações
  • Lista de tabelas
  • Lista de abreviaturas e siglas
  • Lista de símbolos
  • Sumário (deve ser gerado depois de aplicar os estilos).

Configure a galeria de estilos com as seguintes configurações:

Normal. Escolha uma das duas fontes permitidas para elaboração de trabalhos acadêmicos. Tamanho 13. Texto justificado. Espaçamento entre linhas duplo. Espaçamento Antes: 0 pt. Espaçamento depois: 0 pt. Recuo Esquerdo: 0 cm. Recuo Direito: 0 cm. Especial: (nenhum). Fonte de cor preta.

Título 1. Tamanho 14. Texto à esquerda. Espaçamento entre linhas duplo. Espaçamento Antes: 0 pt. Espaçamento depois: 0 pt. Recuo Esquerdo: 0 cm. Recuo Direito: 0 cm. Especial: (nenhum). Fonte de cor preta. Nível 1.

Título 2. Tamanho 13. Texto à esquerda. Espaçamento entre linhas 1,5. Espaçamento Antes: 0 pt. Espaçamento depois: 0 pt. Recuo Esquerdo: 0 cm. Recuo Direito: 0 cm. Especial: (nenhum). Fonte de cor preta. Nível 2.

Associe os títulos de acordo com a especificação da tabela a seguir:

Título e subtítulo Estilo

Referências bibliográficas -


Substitua o desenho de uma tabela por uma tabela criada pelo editor de texto Word.


Atenção: Referências bibliográficas deve estar em uma página separada (Quebra de página).
